# **App Name**: Francis Organics Online

## Core Features:

- Spice Browsing: Browse various organic spice products.
- Product Details: View detailed product information, including ingredients and usage tips.
- Shopping Cart: Add selected items to a persistent shopping cart.
- Simple Checkout: Provide a clear and simple checkout process. No account required.
- Spice Pairing Tool: AI-powered spice pairing tool: Recommend complementary spices based on the selected product, using a tool powered by a large language model.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to evoke nature and organic quality.
- Background color: Desaturated light green (#F0F8F0).
- Accent color: Warm brown (#A0522D) for highlights and calls to action, grounding the earthy tones.
- Body and headline font: 'Literata', serif, for a readable, literary and trustworthy impression.
- Use simple, clean line icons related to spices and organic elements.
- A clean and minimalist layout with clear product categorization for easy browsing.
- Subtle transitions and hover effects to enhance user interaction.
